# sum-time-cli

> Sum up time durations in terminal

## Install

    $ npm install --global sum-time-cli

## Usage

    Usage
      $ sum-time <duration1> <duration2> <duration...>

    Options
      --ms display in miliseconds (no formatting)

    Examples
      $ sum-time 23m 1h
      1h23m
